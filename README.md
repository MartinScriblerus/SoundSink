# SoundSink
## Web-Based Sampler-Sequencer (and more)

### To run locally, 
  * open 2 terminals
  * navigate into the python code, activate virtual env, install requirements.txt, and `fastapi dev main.py`
  * navigate into the Next.ts source code, `npm i` and `npm run dev`
